# Jot it! — Design Brief for Store Assets

## Goal
Create a cohesive icon set and store imagery for the Chrome Web Store listing of **Jot it!**, a friendly, lightweight note‑taking side‑panel extension. Visuals should feel simple, modern, and calm, with a focus on “quick jot” and clarity.

## Logo direction
Preferred concept: **“Jot it!” wordmark where the “i” and “!” are visually related**, with the **“!” as an inverted “i”** (mirrored concept). This can be subtle or explicit.

- Primary logo: **“Jot it!” wordmark**
- Favicon: **“J!”** using the same inverted‑i idea

Please deliver **three distinct logo concepts** that follow this idea (different weights, geometry, or small symbols), so we can pick a direction.

## Visual tone
- Friendly, minimal, and clean
- Avoid overly corporate or playful cartoon styles
- Works on both light and dark backgrounds
- Readable at small sizes

## Required deliverables
### 1) Extension icons (PNG)
Provide final exports in these exact sizes:
- 16x16
- 32x32
- 48x48
- 128x128

Notes:
- Use a square canvas
- Must remain legible at 16px
- Include transparent background

### 2) Chrome Web Store images
Provide ready‑to‑upload PNGs:
- **Store icon**: 128x128 (can be same as extension icon)
- **Large promo tile**: 920x680
- **Marquee promo tile** (optional): 1400x560
- **Screenshots**: 1280x800 (at least 3)

### 3) Favicon
- 32x32 and 48x48 PNG (transparent)
- Built from the “J!” mark that matches the logo concept

## Creative notes / suggestions
- The “inverted i” could be done by mirroring the “i” dot+stem to resemble the “!” dot+stem.
- Consider a subtle **note card** or **side panel** shape behind the mark for app context (optional, but avoid clutter).
- Avoid thin strokes that won’t survive 16px.
- Keep spacing consistent and geometric for readability.

## Brand hints (not strict)
- Colors: soft neutral palette (light gray, slate, charcoal) with a single calm accent (teal, blue‑gray, or muted green).
- Typeface: modern sans, rounded edges OK.
- If you introduce an icon element, it should also work as a monochrome mark.

## What we want from you
Please deliver **three logo directions** with:
- Wordmark “Jot it!”
- The inverted‑i / exclamation relationship clearly demonstrated
- A matching “J!” favicon concept

For each direction, include:
- A short description of the idea
- Color palette (hex values)
- Example of the mark at small size

## File formats
- Source files: SVG or AI (preferred)
- Exports: PNGs listed above
- Include a monochrome version of the logo in black and white

Thank you!
