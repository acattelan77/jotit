# NNNN. <Short, decision-stating title>

Status: Proposed | Accepted | Superseded by NNNN
Date: YYYY-MM-DD

## Context

What situation/problem/tension forced a decision. What options existed.

## Decision

What was decided, stated plainly enough that "what did we decide" has one
unambiguous answer.

## Consequences

What this makes easier, what it makes harder, what it forecloses. Include
the honest downsides — this is what stops a future agent from "fixing" the
downside without understanding it was a known tradeoff.
